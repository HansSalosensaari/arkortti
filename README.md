## Sivun käyttö
Skannaa ylläoleva QR-koodi.
Kun sivu on ladannut ja ruudulla näkyy skanneri, skannaa tämä kuva:
https://github.com/HansSalosensaari/arkortti/blob/main/K%C3%A4yntikorttiScan.png
Käyntikortti on suunniteltu käytettäväksi puhelimella pystyasennossa, mutta se toimii muillakin laitteilla.
